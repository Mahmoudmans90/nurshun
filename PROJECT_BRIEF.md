# Norshen Company Website

## Project Type

Static one-page company website.

## Technologies

- React JavaScript
- Vite
- Tailwind CSS v3
- Motion for React

## No

- Backend
- Database
- React Router
- Docker

## Sections

1. Header
2. Hero
3. About
4. Services
5. Projects with modal
6. Contact
7. Footer

## Requirements

- English-first website
- Responsive on mobile, tablet, and desktop
- Clean professional company design
- Fast performance
- Accessible navigation
- SEO metadata
- Reusable React components
- Content stored in data files

## Deployment

Build using npm run build and upload the contents of dist to Hostinger public_html.
